# Secure Blog - Frontend

Nuxt 4 SPA with Tailwind CSS v4 and shadcn-vue.

## Quick Start

```bash
cp .env.example .env
npm install
npm run dev
```

## Scripts

| Command            | Description                |
|--------------------|----------------------------|
| `npm run dev`      | Start dev server           |
| `npm run build`    | Build for production       |
| `npm run preview`  | Preview production build   |
