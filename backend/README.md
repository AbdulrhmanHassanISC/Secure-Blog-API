# Secure Blog API - Backend

Express.js REST API with MongoDB, JWT auth, role-based access, and file uploads.

## Quick Start

```bash
cp .env.example .env
npm install
npm run dev
```

## Scripts

| Command            | Description                |
|--------------------|----------------------------|
| `npm run dev`      | Start with nodemon         |
| `npm start`        | Start production           |
| `npm run seed:admin` | Seed admin user          |
| `npm run seed:posts`  | Seed 10 sample posts    |
